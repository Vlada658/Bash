# Bash
**Описание задания:**  
Выполнены базовые команды bash для работы с файловой системой, процессами и сетевыми запросами.

**Что было сделано:**  
- Работа с файлами и директориями (создание, удаление, перемещение)  
- Просмотр и редактирование содержимого файлов  
- Поиск данных в файлах и каталогах (grep, find)  
- Работа с процессами (просмотр, завершение)  
- Проверка доступности ресурсов (ping)  
- Выполнение HTTP-запросов через curl (GET, POST)
